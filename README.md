<h1> Weather Web App </h1>
<h2>Overview</h2>
This is a simple weather web application that allows users to check the current weather conditions of different cities. The app is built using HTML, CSS, and JavaScript, and it fetches weather data from the Weather API by Ninjas via Rapid API.

<h2>Features</h2>
Current Weather: Get real-time weather information for a specific city.
Multi-city Weather: View the weather of predefined cities like Shanghai, Boston, Lucknow, and Kolkata.
Responsive Design: The application is designed to work seamlessly on various devices.
Getting Started
Follow these steps to run the Weather Web App locally:

1. Clone the repository:
2. Open the index.html file in your web browser.
3. Input a city name in the search bar or choose a predefined city from the dropdown.
4. Click the "Search" button to fetch and display the weather information.

<h2>API Integration</h2>
This project integrates the Weather API by Ninjas, accessed through Rapid API. To use the API, you need to sign up on the Rapid API website, obtain an API key, and replace the placeholder key in the script.js file.
